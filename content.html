How to use this master:

  • Move repeating html code in new file, for example header.html
  • In main file, like index.html, write
    <jinclude src="header.html" ></jinclude>
  • And insert jQuery and jInclude before </body>
    <script src="js/lib/jquery-2.0.3.min.js"></script>
    <script src="js/jinclude.js"></script>

It will help you to divide page on logic parts and prepare for templating.
